AI Project - "tinc-GPT"
A RAG-based OpenAI chatbot prototype for TINC Tech Incubator, facilitating founders in applying their learnings to business.
Project Details / Background
This ongoing pilot project for Innovation Norway Silicon Valley was to test out the OpenAI API and leverage modern technologies in the coaching of the startups we provide help and guidance for through our programs and events.
Development started in December 2023, and the first version of the project utilized the OpenAI Assistants framework. Then, for the second version of the prototype, I used the OpenAI API to develop a full stack chatbot, leveraging RAG (retrieval augmented generation) with vector embedding techniques, which is yet to be deployed.
